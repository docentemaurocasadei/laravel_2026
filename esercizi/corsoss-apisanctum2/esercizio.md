##creare una api con sancum - 2 esempio

Realizzare un’API autenticata utilizzando Laravel Sanctum che permetta:

di ottenere un token tramite login
di recuperare i dati dell’utente autenticato tramite token

- creare il progetto
- configurare .env
- installare api
- creare la rotta /login
- creare la rotta /users
- creare la logica di validazione dei parametri, autenticazione dell'utente e restituzione del token nella rotta /login
- creare la logica di validazione/autenticazione con il middleware auth:sanctum nella rotta /users che restituisce i dati dell'utente autenticato
